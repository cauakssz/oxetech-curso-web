Já utilizei todos os tipos de CSS.

1- Qual das abordagens é mais prática em um projeto com várias páginas?

R= Estilo externo, criando um arquivo dedicado para CSS.

2- Em qual situação o CSS inline pode ser útil?

R= Na utilização de um estilo que não será reutilizado, como um estilo específico de um único elemento.

3- Por que separar estilo e conteúdo é uma boa prática?

R= Para manter uma melhor organização tanto para quem está usando no momento quanto para as de mais pessoas que queiram ver e entender o código.
